About the bioperl project
The Bioperl Project is an international association of developers of public domain Perl tools for computational molecular biology.

bio.perl.org provides an online resource for modules, scripts, and web links for developers of Perl-based software for life science research. We can also provide web, FTP and CVS space for individuals and organizations wishing to distribute or otherwise make available standalone scripts & code.

Download Latest Stable release
The most up to date release can be found here. Each release passes internal tests. The 0.05 branch is currently the latest stable branch. We are committed to promptly fixing any bugs in the 0.05 release and making patched releases available. The releases can be found in the Comprehensive Perl Archive Network (CPAN). The 0.05 will appear on CPAN in a couple of weeks. CPAN links and info can be found at www.perl.com. The following link will invoke the CPAN multiplexer which will redirect you to a nearby archive site: http://www.perl.com/CPAN/modules/by-module/Bio/.

Volunteers needed
Bioperl is run by volunteers from all over the world, with many types of backgrounds. Any contributions of Perl code, html, documentation writing or code testing are very welcome. We are specifically looking for people to do the following jobs (they should not take more than 1-2 hours a week).
  • FAQ maintainers
  • Script archive manager
  • Cookbook documentation writers
  • Related sites manager
Currently these responsibilities are being shared by the core group and we feel that they can be easy, small projects for people looking to contribute to bioperl.

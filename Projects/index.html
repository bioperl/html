Bioperl Projects
This page is a jumping off point for the current projects underway within the Bioperl effort.

General

A collection of design documents are held in this area. They are coordinated by the overview page, which is where newcomers should start.

  • Overview of Projects at bioperl

  • List of problem areas that bioperl should focus on

  • Status - An overview of the current code and who (roughly speaking) is doing what.

  • Central Distribution - The latest release of the central distribution, with notes on how to download and install it

  • Bioperl Schema - An Overview of the bioperl objects in UML

  • Life Sciences Research Task Force - The contribution of bioperl to this CORBA-based standards effort.

  • Discussions of XML for biology - The start of gathering together resources for XML in biology

Specific

Each of following pages are to specific areas being worked on by people in the bioperl project. Most of these pages are not well documented as we try to gather information on the projects together here.

  • Documentation for the Central Distribution - Documentation of the central distribution

  • Sequence Objects - sequence manipulation and lightweight analyses

  • Sequence Similarity Search - generic sequence similarity searches.
    • Blast Sequence Analysis

  • Sequence DB Access - retrieving sequence objects from databases

  • Sequence Alignments - both dumb and clever representations of alignments and methods for making alignments

  • Sequence Annotation - attaching annotations on sequences

  • Structure Object - a protein structure object

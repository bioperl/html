Bioperl Projects
This page is a jumping off point for the current projects underway within the Bioperl effort.

General

  • Roadmap - An overview of the current code and who (roughly speaking) is doing what.

  • Central Distribution - The latest release of the central distribution, with notes on how to download and install it

  • Documentation for Registered Modules - Documentation of the central distribution

  • Bioperl Schema - An Overview of the bioperl objects in UML

  • Life Sciences Research Task Force - The contribution of bioperl to the Life Sciences Research Task Force

    Specific

    Each of following pages are to specific areas being worked on by people in the bioperl project. Most of these pages are not well documented as we try to gather information on the projects together here.

    • Sequence Objects - both heavy and lightweight versions

    • Sequence Alignments - both dumb and clever representations of alignments and methods for making alignments

    • Blast Sequence Analysis

    • Structure Object - a protein structure object

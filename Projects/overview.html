Overview of Projects
The Bioperl project is a coordinated effort to collect computational methods routinely used in bioinformatics into a set of standard CPAN-style, well-documented, and freely available Perl modules.

bio.perl.org provides a focal point for bioperl activities and offers some central resources, in particular web space and a cvs source repository. In addition, the main developers are keen to coordinate the CPAN-style modules in the cvs repository so that they work together and cover the main problem areas in bioinformatics.

We have attempted to list the areas in bioinformatics for which we feel bioperl can provide a useful solution. These are biased towards biological sequence, but that is the nature of this field in any case.

We are currently focused on modules but a long term aim is to coordinate scripts as well. A place holder for scripts is at Script Central on this server.

This page focuses on the design of modules. The implementation of modules is mainly tracked through the Status page.

Projects that need immediate attention

Looking at the defined areas, we feel the following projects deserve early attention. Many of them already have some code in the Core modules. People who want to help out are encouraged to do so. The best way is to post a message to bioperl discussion list.

  • Sequence Data Manipulation
  • Sequence Similarity Search
  • Sequence DB Access
  • Sequence Alignments
  • Sequence Annotation
The following projects are already being discussed, but are not on the 'most important' list.
  • Biological 3D Structure
  • Biological XML definitions
If people want to open up a new discussion, feel free. You should post to the main bioperl discussion list. If you need web space, just drop a mail to the bioperl webmaster.

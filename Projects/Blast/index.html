Project: Blast Sequence Analysis
The Bioperl Blast module provides an API to the BLAST program, permitting such things as parsing, running, HTML-formatting and manipulations of Blast data.

The Blast package incorporates a modularized version of the webblast script for running Blasts remotely. Support for running Blasts locally via this module is planned but not yet fully implemented.

The Blast package is now included with the central Bioperl distribution. Please see this distribution for installation instructions.

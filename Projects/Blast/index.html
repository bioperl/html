Project: Blast Sequence Analysis
The Bioperl Blast module provides an API to the BLAST program, and thereby permits the parsing, running, HTML-formatting, and general manipulation of Blast data by simple method calls on Perl objects. It does not implement the Blast algorithm itself, relying on external applications for this compute-intensive operation.

The Bioperl Blast distribution is now included with the central Bioperl distribution. Please follow this link for installation instructions.

DOCUMENTATION:

In addition to Blast.pm, the Bioperl Blast package contains several separate but integrated Perl modules:

  • Sbjct.pm
  • HSP.pm
  • HTML.pm
  • Webblast.pm
  • LocalBlast.pm

Webblast.pm is a modularized version of the webblast script for running Blasts remotely. Support for running Blasts locally is provided by the LocalBlast.pm module, which is at present just a skeleton. Ultimately, this module will require some amount of customization for different local installations of the Blast executables.

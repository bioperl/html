Project News

[December 1998]

  • Maintenance release bioperl-0.04.2 is available.
  • Ewan Birney & Matthew Pocock have made their prEMBL modules available from the Sanger Centre. prEMBL is a set of Perl modules designed for the easy parsing and manipulation of EMBL-format type databases.
  • Maintenance release bioperl-0.04.1 is available.
  • 3D Biomolecular Structure Handling - Eser Ayanoglu and others at Stanford have issued a request for comments.
  • bioperl-0.04 lives! Officially updated with new code, new documentation and test suit. Follow the threads on the bioperl and bioperl-guts lists.
  • Hardware heads may be interested in pictures of our server setup.

[November 1998]

  • Lincoln Stein has released version 1.10 of BoulderIO. Read his annoucement here.

  • The first 'official' bioperl release will be announced shortly. We are extremely interested in feedback regarding installation/compilation experiences.

[October 1998]

  • Ewan has released a Wise2 beta that is useable via Perl and XS. Info on the Perl port is (here).
  • Bioperl examples/ directory started. First step towards some "cookbook" style documentation.
  • SteveC has updated RestrictionEnzyme.pm
  • Milestone: - bioperl now builds from a central Makefile.PL (including the C XS extensions). We are getting closer and closer to our goal of being able to build public distributions directly from our central source tree. Check the the guts-list for the details.

[September 1998]

  • Ewan announces 1st version of Protein Smith-Waterman interface in Bio::Tools::pSW (uses XS)
  • Ian announces 1st versions of Bio::SeqFeature and Bio::SeqFeatureParser
  • SteveC has updated Bio::Tools::Blast
  • Andewq Dalke has a Python/Perl package for parsing PDB files
  • CORBA LSR/DSIG press release

[August 1998]

  • Announce: PaintBlast.pm - parsing and graphical representation of a Blast search output
  • Announce: CheckSeq.pm - checking coherency between a given sequence and a selected Blast program
  • Steve Chervitz has updated his Bioperl Blast.pm module
  • bio.perl.org is born!
  • A bioperl poster was presented at ISMB'98 and OiB'98.

[July 1998]

  • SteveC's ISMB'98 Meeting minutes & summary
  • ANNOUNCE: NHGRI::Blastall.pm (alpha)
  • ANNOUNCE: PubMed Monitor (alpha)

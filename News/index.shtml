Project News

[December 2000]

[October - November 2000]

  • Bioperl 0.6.2 has been released. It is mostly a bugfix release with very few functional changes.This should be the final 0.6 series release. The main branch of our current CVS repository (bioperl-live) contains significant new code and functionality that is *NOT* part of the 0.6.2 release.

  • We are working on coordinating a Bioperl 0.7 release that will merge all the new features into the main distribution. We estimate that Bioperl 0.7 will come out by year end. Scanning the bioperl-l archives is a good way to see what new code and features have been added.

[August / September 2000]

  • Pictures from BOSC'2000 are online. We are working on getting the presentations and proceedings online as well. Send comments, corrections and additional pictures to dag@sonsorol.org.

  • The Bioperl website is in the process of becoming Wiki-enabled. We hope the collaborative editing and authoring features will keep the website more topical, accurate and up-to-date. To see a Wiki site in action check out the EnsemblDocs Wiki.

  • Peter van Heusden is coordinating the release of Bioperl 0.6.2 which is a bug fix release of Bioperl 0.6.1. This should be available in September. The main CVS branch of Bioperl contains significant new code and functionality that will *NOT* be part of the 0.6.2 release. We are working on coordinating a Bioperl 0.7 release that will merge all the new features into the main distribution. We estimate that Bioperl 0.7 will come out in October or November. Scanning the bioperl-l archives is a good way to see what new code and features have been added.

  • Bioperl has been added to the FreeBSD ports tree! Thanks to Johann Visagie for putting this together.

  • A new sister site biocorba.org was launched during BOSC'2000. Jason Stajich is leading the effort.

  • Our Bioinformatics Open Source Conference (BOSC'2000) was a success with more than 220 attendees. Pictures and presentations will be up shortly.

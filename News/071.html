The module Bio::Tools::pSW has been checked into the cvs
system.

To use it you need to install the C 'engine' which is in
Bio::Compile::SW. At the moment the root level Makefile
does not install this (it doesn't work at all for me so... ;)).
This is problem is biggest outstanding problem for this
module.

You can install yourself by cd'ing into Bio/Compile/SW, perl Makefile.PL,
make,make install.

The engine goes into bp_sw:: namespace in perl.

At the moment all pSW can do is produce a 'pretty' alignment into
a file. Here is an example script

#!/usr/local/bin/perl

use Bio::Seq;
use Bio::Tools::pSW;

$seq1 = Bio::Seq->new ( '-file' => 'road.pep');
$seq2 = Bio::Seq->new ( '-file' => 'roac.pep');

$factory = new Bio::Tools::pSW(); # default values

$factory->align_and_show($seq1,$seq2,STDOUT);

Soon(ish) it will build Bio::SimpleAlign objects and Bio::UnivAlign
Objects.

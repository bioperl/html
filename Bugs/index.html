Bioperl Bugs

Reporting Bugs

The Bioperl developers are keen on tracking down and exorcising bugs from all code included in the central distribution. If you have a bug to report, you have two options:
  • Use the bug-tracking web interface (Recommended!)

  • Send e-mail to:   bioperl-bugs@bio.perl.org

Bug Information

You can view the status of reported bugs using the bug-tracking web interface either as a guest or a registered user.

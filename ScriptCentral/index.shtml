Molecular Biological Software at the Hospital for Sick Children, Toronto
Author: Eddy Wong
URL: http://www.genet.sickkids.on.ca/bioinfo_resources/software.html
Description:
A number of general perl scripts for sequence data mining, including webblast -- a perl script which allows you to BLAST sequences at NCBI via the web. The bioperl code for webblasting was orginially based on one of these scripts

Will Fischer's Scripts
Author: Will Fischer
URL: http://sunflower.bio.indiana.edu/~wfischer/Perl_Scripts/
Description:
Standalone perl scripts for searching GenBank files by pattern, translating nucleotide sequences to amino-acids (DNA to protein), making DNA alignments from amino-acid alignments, converting NEXUS files (from PAUP or MacClade) into FASTA or MEGA format, extracting the translations supplied by GenBank from CDS features in Genbank file

Paintblast.pm
Author: Alessandro Guffanti
URL: http://hercules.tigem.it/Biomodules.html
Description: Parses a Blast output and produces a GIF image with a schematic representation of the resulting alignments. Module is still experimental.
REQUIRES: needs Lincoln Stein's GD module. GD module requires Tom Boutell's graphic libgd library.

CheckSeq.pm
Author: Alessandro Guffanti
URL: http://hercules.tigem.it/Biomodules.html
Description: Module examines an input sequence, reformats it in FastA format and determines whether it is a DNA or Protein sequence. It then checks the coherency between the sequence type and the chosen blast program (blastn, blastp, tblastn, blastx, tblastx). The module was created in order to avoid potential problems for users of web-based Blast-based services.
REQUIRES: module uses the fmtseq program from J.R.Knight's SeqIo package.

TBOB
Author: Pierre Rioux
URL: ftp://megasun.bch.umontreal.ca/pub/tbob
Description: Text based BLAST output Browser. Output from the BLAST programs can be difficult to read, especially when there are many search results to be analyzed, each potentially with many matches with sequences in the database. TBOB solves this problem by only showing parts of the BLAST output at any time, letting the user jump from one place to another using simple one-keystoke commands. TBOB also permits retrieval of sequences from the NCBI retrieve e-mail server, so that interesting sequences can be obtained for further analysis.

MOLPHY
Author: Jun Adachi
URL: ftp://sunmh.ism.ac.jp/pub/molphy
Description: MOLPHY is a program package for MOLecular PHYlogenetics. PROTML is a main program in MOLPHY for inferring evolutionary trees from PROTein sequences by using the Maximum Likelihood method. The MOLPHY package includes a number of useful Perl Utilites for DNA/Protein sequence manipulation.

PARBOOT
Author: Pierre Rioux
URL: ftp://megasun.bch.umontreal.ca/pub/parboot
Description: PARBOOT is a parallel implementaion of the bootstrapping function available in many of the PHYLIP phylogenetic analysis programs. The PARBOOT application splits a multiple-dataset input file into its independent datasets, and processes these datasets in parallel on multiple hosts (or one host with multiple CPUs). This means that the speed of a bootsptrap analysis can be approximately improved in proportion to the number and power of CPUs available to the user.
REQUIRES: rsh, rcp, PHYLIP

SLBC - Search Launcher Batch Client
Author: Randall F. Smith
URL: http://dot.imgen.bcm.tmc.edu:9331/downloads/software/batch_client.html
Description: The SLBC will read in multiple input files, perform a specified search (e.g., BLAST, BEAUTY) on each sequence (on at a time) then store the result files on the user's system as individual HTML documents. The search results can then be viewed using an HTML viewer. The search results will be the same as if the search was performed from the Baylor College of Medicine's Search Launcher Web Pages.

GEANFAMMER - GEnome ANalysis and Protein FAMily MakER
Author: Jong Park
URL: http://www.mrc-lmb.cam.ac.uk/genomes/geanfammer.html
Description: Geanfammer has been developed for the analysis of most of the complete bacterial genomes announced since 1995. It summarizes the whole procedure of preparing statistically and biologically more relevant protein (sequence) duplication modules before any more biological analysis like structure and fucntion assignment.

Freely available perl code for the life sciences {under construction}
 
A call for help: Creating and keeping this index organized and current is a time consuming task. Help us out by letting us know if you spot a dead link, errors or know of some new software that has been released. Anyone wishing to help maintain or work on this list shoud contact Chris Dagdigian.
 
Sanger Centre Modules
Author: Ewan Birney, others.
URL: http://www.sanger.ac.uk/Software/PerlModule/
Description: Sequence and Alignment modules, EMBL modules, Feature Table & GFF modules, ACEdb modules.

Will Fischer's Scripts
Author: Will Fischer
URL: http://hyrax.bio.indiana.edu:81/~wfischer/Perl_Scripts/
Description:
Standalone perl scripts for searching GenBank files by pattern, translating nucleotide sequences to amino-acids (DNA to protein), making DNA alignments from amino-acid alignments, converting NEXUS files (from PAUP or MacClade) into FASTA or MEGA format, extracting the translations supplied by GenBank from CDS features in Genbank file

Lincoln Stein's Modules
Author:Lincoln Stein
URL: http://stein.cshl.org/
http://stein.cshl.org/software/boulder/
http://stein.cshl.org/AcePerl/
http://stein.cshl.org/software/boulder/
http://stein.cshl.org/WWW/software/
Description: Author of the famous CGI:: modules also has code for the Boulder I/O format and perl interfaces to the ACEdb database.

Jong Park's Perl Subroutine Library - Bio.pl
Author:Jong Park, others.
URL: http://cyrah.med.harvard.edu/~jong/Bio.pl.html
Description: A huge library of bio-related perl subroutines. A great place to find bits of useful code to add into your own programs.

Ian Korf's Code & Bio::GSC Modules
Author: Ian Korf
URL: http://sapiens.wustl.edu/~ikorf/bioinformatics.html Description: Ian has made public the Bio::GSC modules that the WashU Genome Sequencing Center uses. He also has a standalone Blast parser called BP2.pl.

Roland Walker's SEALS system
Author: Roland Walker
URL: http://www.ncbi.nlm.nih.gov/Walker/SEALS/
Description: SEALS (A System for Easy Analysis of Lots of Sequences) is a perl software package expressly designed for large-scale research projects in bioinformatics.

Paintblast.pm
Author: Alessandro Guffanti
URL: http://hercules.tigem.it/Biomodules.html
Description: Parses a Blast output and produces a GIF image with a schematic representation of the resulting alignments. Module is still experimental.
REQUIRES: needs Lincoln Stein's GD module. GD module requires Tom Boutell's graphic libgd library.

CheckSeq.pm
Author: Alessandro Guffanti
URL: http://hercules.tigem.it/Biomodules.html
Description: Module examines an input sequence, reformats it in FastA format and determines whether it is a DNA or Protein sequence. It then checks the coherency between the sequence type and the chosen blast program (blastn, blastp, tblastn, blastx, tblastx). The module was created in order to avoid potential problems for users of web-based Blast-based services.
REQUIRES: module uses the fmtseq program from J.R.Knight's SeqIo package.

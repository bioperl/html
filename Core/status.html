Bioperl Development Status


Where we are. What we have. Where we are going.

The bioperl sequence object has been integrated into a general framework developed by Steve Chervitz. The current framework has since expanded and is comprised of a set of integrated modules The core distribution is now available worldwide via CPAN. Our modules can be found in the Bio/ directory.

Design
The overall design of the core modules is (roughly) coordinated from the Projects overview page. This page focuses on the Perl module code which is already in the CVS system or expected to be eventually.

Current Efforts
Bioperl is now running smoothly from a central source code repository. The current 0.05.1 release came out on 30 June 1999. The 0.05 branch has been frozen (bug fixes only) while we prepare for the next core release.

Our first Bioperl Workshop held right before ISMB-99 in Heidelberg, Germany was a great success and we have attracted some new developers. We expect the next major bioperl release to occur in December.

Code Status and Committments
We have changed our versioning scheme, the next bioperl release will be 0.6 not 0.06!

PreSeq.pm has been replaced by Seq.pm and will become our general flexible and lightweight sequence object. Parse.pm is no longer being used by core bioperl code. We will also be moving all sequence formatting and parsing code out of the sequence object and transitioning over to the SeqIO/* modules

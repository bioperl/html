Bioperl is a collection of "standard" CPAN type modules so installation should be fairly easy and straightforward. Grab either the bioperl-06.0.tar.gz file or the bioperl-06.0.zip file from this site via the HTTP or FTP links above.

The latest distribution, bioperl-06.0 should still be considered a for-unix-only alpha release. We use the package for real work at our sites, but if you are considering using bioperl in production work, we would recommend both heavily testing the package on your site and posting a mail to discussion list to hear whether anyone else has had problems with the functionality you want to use.

We are particularly interested in hearing about how the installation process works on different platforms and operating systems. General comments, suggestions and installation war stories should be sent to the developers email-list vsns-bcd-perl-guts@lists.uni-bielefeld.de. Bug reports should be submitted via our web-based reporting system at http://bio.perl.org/bioperl-bugs/ or by sending email to bioperl-bugs@bio.perl.org.

After obtaining the distribution file, unpack it and read the installation instructions in README file.

The distribution comes with its own internal documentation in the perl documentation format (pod). This documentation is has been translated into HTML links available below. The best starting point for people learning about bioperl is the bioperl.pod documentation, followed by the Bio::Seq.pm documentation.

In addition to embedded POD-format documentation, there is also an examples/ directory .

Current Efforts
General housekeeping efforts related to moving our code to a common source repository are underway. This includes getting bioperl deveopers up to speed regarding version control via CVS, building test suites, documention repositories and perl module Makefiles. The goal is to be able to build "official" bioperl module distributions off of one common source tree.

Code status & Committments
PreSeq.pm will be replaced by Seq.pm and will become our general flexible and lightweight sequence object. Ian Korf is integrating the functionality of his GSC::SeqFeature and GSC::Entry into the bioperl framwork. Ewan Birney is working on the following:

  • SimpleAlign.pm testing
  • Histogram package from Sean Eddy as an XS extension
  • SmithWaterman as an XS extension
  • ReadSeq format interconverter as an XS extension

Possible new projects
Several people are interested creating a protein structure object. Discussions on the basic design are underway on our mailing lists. Comments, proposals and participation are welcome.

Current Efforts
General housekeeping efforts related to moving our code to a common source repository are underway. This includes getting bioperl deveopers up to speed regarding version control via CVS, building test suites, documention repositories and perl module Makefiles. The goal is to be able to build "official" bioperl module distributions off of one common source tree.

We expect to release a preliminary central distribution very shorty as we now have all our code building off a central Makefile.PL

Code status & Committments
PreSeq.pm has been replaced by Seq.pm and will become our general flexible and lightweight sequence object. Parse.pm is no longer being used by core bioperl code.

To-do list

  • Bio::SeqFeature and Bio::Entry - The Bio::Entry is the 'heavy weight' sequence object which is more like a representation of a Genbank or EMBL flat file complete with feature table. Ian Korf is working on these objects.
  • Bio::Structure - A number of people have been talking about a Structure object (probably an object heirarchy).
  • Bio::SearchDist - A perl wrapper for Sean Eddy's Extreme Value and Gaussian fitting code (already provided in the bp_sw XS extension). Ewan Birney is working on this.
  • Bio::Tools::dAlign - DNA alignments like pSW provided in a perl wrapper. Ewan Birney is working on this.
Steve Chervits, the bioperl coordinator is maintaining and improving the Bio::Tools::Blast and Bio::Root code.

About the bioperl project

The Bioperl project is a coordinated effort to collect computational methods routinely used in bioinformatics into a set of standard, well-documented, and freely available Perl modules. Perl provides unparalleled support for many tasks common in bioinformatics yet there are no standard Perl modules for biology. We hope to help fill this void.

The bioperl project is open to all and we invite suggestions and participation from the larger bioinformatics community.

bio.perl.org, the central site for our efforts became live in August 1998. Many features of the site are still in actively being constructed. For additional information and links, check out the Bielefeld Bioperl page.
